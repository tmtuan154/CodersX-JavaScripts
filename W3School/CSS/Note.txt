1. Syntax
 - selector: h1, p, ...
 - decleration: {property: value}
 
2. Selector
 - id selector: using by '#id'
 - class selector: using by '.className'. Apply for all HTML elements with class is className
 - element.class: using when you want to style with species element and class. Ex: p.center: in this example only <p> element with class = 'center'
 - HTML can contain many class for element. <div class= 'a b'>...
 - You can group selector by using h1, h2, p {property: value, ...}

3. How to use
 - Inline
 - Internal
 - External: <link rel="stylesheet" href="mystyle.css">

8. Margin
 - Margin used to create space arounf elements, outside of any defined borders
 - Margin also have margin-top -left -bottom - right
 - Margin shorthand: margin: top right bottom left

9. Pading
 - Similar with margin: pading create space around an element's content, inside of any defined border

10. Height, width
 - Height, width don't contain border, pading and margin
 
11. Box model
 - Margin -> Border -> Pading -> Content
 - ex: div {
		  width: 320px;
		  padding: 10px;
		  border: 5px solid gray;
		  margin: 0;
		}
 - total width is 350px
